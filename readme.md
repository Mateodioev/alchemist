## Installation 

_Github_:

```bash
git clone https://github.com/Mateodioev/alchemist.git
cd alchemist
composer install
```

_Composer_ (**recommended**):

```bash
composer require mateodioev/alchemist

# if not work try:
composer require mateodioev/alchemist:dev-master
```
